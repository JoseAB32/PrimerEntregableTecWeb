Elementos De HTML
Nro De Elemento Proposito Ejemplo
1 Div Elemento generico de tipo contenedor para contenido de flujo <div>
<p>Hola Mundo</p>
</div>
2 h1 Elemento de encabezado de nivel 1, existen 6 niveles del h1 al h6, es recomendado solo usar un h1 por pagina <h1>Hola Mundo</h1>
3 p Elemento de párrafo <p>Este es un párrafo</p>
4 span Elemento en línea genérico, para agregar estilos en medio de textos <span>Texto en línea</span>
5 strong Elemento de texto en negrita <strong>Texto en negrita</strong>
6 em Elemento de texto en cursiva <em>Texto en cursiva</em>
7 mark Elemento de texto resaltado <mark>Texto resaltado</mark>
8 del Elemento de texto tachado <del>Texto tachado</del>
9 ins Elemento de texto subrayado <ins>Texto subrayado</ins>
10 sub Elemento de texto subíndice <p>Mira asi se ve un<sub>Texto subíndice</sub></p>
11 sup Elemento de texto superíndice <p>Mira asi se ve un<sup>Texto superíndice</sup></p>
12 img Elemento para insertar una imagen <img src="ruta/a/imagen.jpg" alt="Descripción de la imagen"> <img src="direccion en linea" alt="Descripción de la imagen">
13 quote Elemento para citas cortas en línea <q>Esta es una cita</q>
14 blockquote Elemento para citas en bloque <blockquote>
<p>Esta es una cita en bloque</p>
</blockquote>
15 small Elemento para texto de menor tamaño <small>Texto pequeño</small>
16 cite Elemento para citar una referencia <cite>Referencia citada</cite>
17 code Elemento para mostrar código <code><div>
int a = 5;
int b = 10;
int suma(a, b){
return a + b;
}
cout << suma(a, b);
</div></code>
18 samp Elemento para mostrar salida de un programa <samp>Error: 404 Not Found</samp>
19 kbd Elemento para mostrar entrada del teclado <kbd>Ctrl + C</kbd>
20 pre Elemento para mostrar texto preformateado <pre> 
                        .-""""""-.
                      .'          '.
                     /   O      O   \
                    :           `    :
                    |                |   
                    :    .------.    :
                     \  '        '  /
                      '.          .'
                        '-......-'
</pre>
21 address Elemento para mostrar información de contacto <address>
<p>Nombre: Juan Perez</p>
<p>Email: juan.perez@example.com</p>
</address>
22 entity Elemento para mostrar caracteres especiales &amp; &
&lt; <
&gt; >
&quot; "
&apos; '
23 a Elemento para crear un hipervínculo <a href="https://www.example.com">Enlace a Example</a>
24 br Elemento para un salto de línea <br>
25 wbr Elemento para un posible punto de quiebre en una línea de texto <wbr>
ELEMENTOS SEMANTICOS
26 header Elemento para definir el encabezado de una sección o página <header>
<h1>Encabezado</h1>
<p>Este es el encabezado de la página</p>
</header>
27 footer Elemento para definir el pie de página <footer>
<p>Este es el pie de página</p>
</footer>
28 article Elemento para definir contenido independiente y auto-contenido <article>
<h2>Título del artículo</h2>
<p>Contenido del artículo</p>
</article>
29 section Elemento para definir una sección de contenido <section>
<h2>Título de la sección</h2>
<p>Contenido de la sección</p>
</section>
30 nav Elemento para definir un bloque de enlaces de navegación <nav>
<ul>
<li><a href="#">Enlace 1</a></li>
<li><a href="#">Enlace 2</a></li>
</ul>
</nav>
31 aside Elemento para definir contenido relacionado pero independiente del contenido principal <aside>
<p>Este es un contenido relacionado</p>
</aside>
32 main Elemento para definir el contenido principal del documento <main>
<h2>Contenido Principal</h2>
<p>Este es el contenido principal de la página</p>
</main>
ELEMENTOS DE TABLA
33 table Elemento para definir una tabla <table>
<tr>
<td>Celda 1</td>
<td>Celda 2</td>
</tr>
</table>
34 tr Elemento para definir una fila en una tabla <table>
<tr>
<td>Celda 1</td>
<td>Celda 2</td>
</tr>
</table>
35 td Elemento para definir una celda en una tabla <table>
<tr>
<td>Celda 1</td>
<td>Celda 2</td>
</tr>
</table>
36 th Elemento para definir una celda de encabezado en una tabla <table>
<tr>
<th>Encabezado 1</th>
<th>Encabezado 2</th>
</tr>
</table>
37 thead Elemento para agrupar el contenido del encabezado en una tabla <table>
<thead>
<tr>
<th>Encabezado 1</th>
<th>Encabezado 2</th>
</tr>
</thead>
</table>
38 tbody Elemento para agrupar el contenido del cuerpo en una tabla <table>
<tbody>
<tr>
<td>Celda 1</td>
<td>Celda 2</td>
</tr>
</tbody>
</table>
39 tfoot Elemento para agrupar el contenido del pie en una tabla <table>
<tfoot>
<tr>
<td>Pie 1</td>
<td>Pie 2</td>
</tr>
</tfoot>
</table>
Hay muchos mas elementos en html, pero estos son algunos de los mas importantes
